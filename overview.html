JavaFX Software Development Kit
Version @JAVADOC_VERSION@ API Specification

The JavaFX API is composed of the following groups of modules:
  • Core JavaFX Modules: These APIs are in modules whose names start with javafx. They are part of the core JavaFX platform.
  • JavaScript Object: These APIs are in the jdk.jsobject module.
  • Incubator Modules: These APIs are in modules whose names start with jfx.incubator. They are experimental APIs subject to modification or removal in subsequent versions of JavaFX.
